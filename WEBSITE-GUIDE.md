## 🛠️ Customization Guide

### Colors
1. Navigate to **Appearance > Customize**
2. Go to **Colors** section
3. Adjust theme colors as needed
4. Changes apply in real-time

### Typography
1. Access **Appearance > Customize**
2. Select **Typography** options
3. Choose fonts and sizes
4. Preview changes instantly

### Layout
1. Open **Appearance > Customize**
2. Find **Layout** settings
3. Adjust container widths
4. Set content areas

### Content
1. Use the **Block Editor** for pages/posts
2. Apply **Block Styles** for enhanced appearance
3. Use **Block Patterns** for quick layouts
4. Customize with **Custom CSS** if needed

## 🔧 Technical Details

### File Structure
```
themes/obsidian/
├── templates/          # Page templates
├── parts/             # Template parts
├── assets/
│   ├── css/          # Stylesheets
│   └── js/           # JavaScript files
├── includes/         # PHP includes
└── style.css         # Main stylesheet
```

### CSS Architecture
- **style.css**: WordPress theme header
- **style-main.css**: WordPress integration styles
- **dynamic.css**: Theme-specific styles
- **block-editor.css**: Editor-specific styles
- **admin.css**: Admin panel styles
