Getting Started

Setting up Obsidian is straightforward. Follow these simple steps to get your website up and running:

  1. Install the Theme: Upload and activate Obsidian through your WordPress admin panel
  2. Customize Colors: Use the WordPress Customizer to set your brand colors
  3. Configure Typography: Choose fonts that match your brand identity
  4. Create Content: Start building pages using the powerful block editor
